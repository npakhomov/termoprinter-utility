# ThermoPrinterTool

Windows-утилита для пошагового подключения термопринтеров:

- ATOL DD340
- ATOL BP41
- TSC DA220
- BIXOLON XD3-40d

## Как запустить готовый exe

Открой файл:

`bin\Release\net10.0-windows\ThermoPrinterTool.exe`

## Как открыть проект в Visual Studio

Открой файл:

`ThermoPrinterTool.slnx`

После открытия нажми `F5`, чтобы запустить приложение из Visual Studio.

## Новый сценарий работы

1. Выбери модель принтера и установленный принтер Windows.
2. Нажми `Проверить USB`.
   Рядом появится цветной статус: `Подключен` или `Не подключен`.
3. Нажми `Найти свободный IP`.
   Программа проверит адреса в текущей сети и подставит первый адрес, который не отвечает на ping.
4. Нажми `Скачать/открыть утилиту`.
   Пока кнопка открывает страницу утилиты. Позже можно добавить запуск твоих локальных файлов.
5. После назначения IP в фирменной утилите нажми `Проверить ping`.
   Рядом появится цветной статус: `Подключен` или `Не подключен`.
6. При необходимости нажми `Создать TCP/IP порт`, чтобы добавить сетевой порт Windows.
7. Нажми `Тестовая печать`, чтобы проверить печать через выбранный драйвер.

## Важное про свободный IP

Сейчас свободный IP определяется по принципу: адрес не отвечает на ping. Это хороший первичный способ для настройки принтера, но не абсолютная гарантия, потому что некоторые устройства могут блокировать ping.

## Важное про автоматическую настройку IP

Программа пока не заходит внутрь фирменных утилит и не назначает IP сама.

Это можно добавить позже, когда будут сами утилиты:

- если у утилиты есть командная строка, подключим тихую настройку;
- если есть конфигурационные файлы, будем менять их автоматически;
- если доступен web-интерфейс принтера, можно попробовать автоматизировать настройку через него;
- если утилита только графическая, можно сделать запуск с подсказками или отдельный сценарий под конкретную модель.
